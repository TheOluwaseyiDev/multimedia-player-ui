# Multimedia Player UI

## Overview
This project is a simple multimedia player interface built using HTML5, Demonstrates structured layout, semantic HTML, and embedding audio and vido content.

## Features
-HTML5 '<audio>' and '<Video>' elements
-Subtitles support with '<track>'
-Structured content with '<section>' and headings
-Transcript section for accessibility and context

## Tools Used
-HTML5.
